# README

### Dependencies
* [PostgreSQL](http://postgresapp.com/)
* Ruby 2.3.0
* Rails 4.2.6
* [Vue.js](https://vuejs.org/)

### Getting Up And Running

* Clone down repo: `$ git clone https://github.com/kristinruben/netivot.git`
* Install gems: `$ bundle install`
* Create the database:
```
$ rake db:create
$ rake db:schema:load
$ rake db:seed
```
* Start the server: `$ rails s`
